## 🚀 如何使用这个仓库

### 1. 克隆仓库（其他人使用）
```bash
# 使用 HTTPS
git clone https://github.com/xianbuzhaoji-jpg/learningspringboot.git

# 或使用 SSH（需要配置 SSH 密钥）
git clone git@github.com:xianbuzhaoji-jpg/learningspringboot.git
```

### 2. 运行项目
```bash
cd learningspringboot

# Windows
run.bat

# Linux/Mac
./mvnw spring-boot:run
```

### 3. 访问应用
- 首页: http://localhost:8080/
- API 文档: http://localhost:8080/api/users
- 模板页面: http://localhost:8080/greeting
